# DESIGN.md

## 产品定位
AdsCraft - Facebook广告决策引擎。不是教程,是决策工具。

## 设计思考

### 气质与意象
- **关键词**: 专业、简洁、高效、信任
- **具象场景**: 商务人士在办公桌前,快速获取决策建议,屏幕上是清晰的配置表格和简洁的界面

### 视觉策略
- 摄影方向: 干净的商务场景,避免过度复杂的视觉元素
- 图形语言: 简洁的卡片式布局,清晰的表格展示

### 配色方案
- 主色调: 蓝色系 (blue-600, blue-50)
- 背景: 渐变背景 (从 blue-50 到 indigo-100),营造专业氛围
- 文字: 灰色系 (gray-900, gray-600, gray-500),保证可读性
- 强调色: 紫色系用于付费升级提示 (purple-50, purple-800)

### 字体排版
- 使用系统默认字体 (antialiased)
- 标题层级清晰: 4xl/6xl (Hero), 3xl (页面标题), 2xl (副标题), xl (章节标题)
- 段落文字使用 text-gray-600 保持可读性

### 动效与交互
- 过渡动画: hover:shadow-lg, hover:border-blue-500, hover:bg-gray-50
- 交互反馈: 按钮状态变化 (disabled/enabled), 卡片点击响应
- 保持简洁,不添加过多动画以免分散注意力

### 页面结构
- 首页: Hero区 + 路线选择卡片 + 三层价值展示 + Footer
- 问答页: 进度指示器 + 问题卡片 + 选项列表
- 方案页: 配置表格 + 升级提示
- 登录/注册页: 简洁的卡片式表单

### 设计禁忌
- 不要使用过于炫酷的动画效果
- 不要使用过多的颜色和渐变
- 不要添加不必要的装饰元素
- 保持界面简洁专业,符合商务决策工具的气质
