## 更新 1：Unknown Campaign 显示优化

**问题：**
当 campaign 名称缺失时，显示 "Unknown Campaign"，用户体验不好。

**解决方案：**
当 campaign 名称为空或 "Unknown Campaign" 时，显示 "[Platform] Analysis"。

**示例：**
- TikTok 广告 → 显示 "TikTok Analysis"

- Google 广告 → 显示 "Google Analysis"

**修改位置：**

**实现逻辑：**
```

  displayName = `${platform} Analysis`
```

## 更新 2：平台识别功能

**问题：**
用户可能把 Facebook 截图上传到 TikTok 模块，需要检测并提示。

**解决方案：**
在截图识别的 prompt 中增加平台判断任务，让 AI 同时识别平台。

**实现方式：**
修改截图识别的 prompt，增加：
> "请判断这是哪个广告平台的截图（TikTok/Facebook/Google），并返回平台标识。"

**AI 返回格式：**
```json
{
  "platform_detected": "tiktok",
  "campaign_name": "Summer Sale 2026",
  "data": { ... }
}
```

**前端逻辑：**
```
if (platform_detected !== platform_selected) {
  showError(`检测到这是 ${platform_detected} 广告，你选择的是 ${platform_selected}`)
}
```

**判断标准（AI 通过以下特征识别）：**
- 视觉：Logo、UI 风格、导航栏文字
- 数据：字段名称、数据格式

**修改位置：**
截图识别的 prompt（调用 GPT-4o-mini 的地方）
